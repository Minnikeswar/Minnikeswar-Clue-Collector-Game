Welcome to the Memory Game! This is a fun and challenging game that tests your memory skills. In this game, you will be presented with a 3X4 grid of cards, with each card containing a hidden image.

Your task is to reveal two cards at a time and match the images on them. You will be scored based on the number of attempts it takes you to match all the cards correctly. This game is a great way to improve your memory and concentration skills, so let's get started!
